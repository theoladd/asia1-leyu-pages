# asia1-leyu-pages

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的 GitHub 仓库。  
仓库中的页面内容彼此独立，不针对任何特定域名或具体网站，仅作为通用 HTML 页面集合的存储与展示用途。

## 目录说明

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 存放各独立 HTML 页面的目录
│   ├── example1.html
│   ├── example2.html
│   └── ...
└── README.md           # 本说明文件
```

- `pages/` 目录为主要内容存放位置，每个 HTML 文件为一个独立页面。
- 根目录下的 `index.html` 可选，用于提供导航或概览。

## 页面归档说明

本仓库中的 HTML 页面均为静态内容，无需后端支持。  
页面可包含样式（CSS）、脚本（JavaScript）及资源引用（如图片、字体等），建议将外部资源托管于可靠 CDN 或与页面文件一同提交。

归档的页面可能来自不同时期或不同用途，彼此之间无关联性。  
每个页面均以文件名作为唯一标识，请勿修改已有文件名以免影响引用链接。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面，请确保新页面不包含恶意代码、不违反平台服务条款。
- 若发现页面存在兼容性问题、链接失效或内容错误，请提交 Issue 说明具体页面及问题描述。
- 本仓库不保证所有页面长期可用，也不承担页面内容引发的任何责任。
- 建议在提交页面时附带简要描述（可在 commit message 中说明页面用途或来源）。
